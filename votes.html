⚙️ Setup Required: This app uses JSONBin.io as a free database.
  1. Go to jsonbin.io → sign up free → create a Bin with initial content {"polls":[],"votes":{}}
  2. Copy your Bin ID and API Key (Master Key or Access Key)
  3. Click Admin above and enter your password (admin123 by default, change it in the code)
  4. Paste your Bin ID and API Key in the Admin Setup panel below
💡 Alternatively: If you change the ADMIN_PASSWORD variable in this file's script, use your chosen password.
